art i don't like

art i don't like is an art anti-recommender. We will make sure to find something you will not find appealing.

How, you ask? Using state-of-the-art Machine Learning algorithms and a vast database of paintings and sculptures, we can guarantee you will not enjoy the artworks we will recommend.
